# 山西焦煤投资分析框架
> 目标：通过量化指标，判断钢铁/焦煤周期位置，确认反转信号，辅助仓位决策

---

## 一、核心认知前提

| 认知点 | 说明 |
|--------|------|
| 焦煤跟谁走 | **跟钢铁走**，不跟动力煤。核心驱动是铁水日产量 |
| 焦煤弹性 | 比螺纹钢更大。钢铁涨2.5倍，焦煤历史上涨6倍（2016→2022） |
| 焦煤滞后性 | 焦煤反转永远**滞后钢价1-2个季度**，但涨幅更大 |
| 动力煤关系 | 短期情绪传染（资金跨品种联动），不是基本面联动，属于噪音 |
| 当前周期位置 | 下行第四年（2021→），政策底可能已现，价格底和盈利底滞后 |

## 三、反转信号系统（按出现顺序）

### 第一层：先行信号（反转前3-6个月）

这些指标先于价格出现，是你提前布局的依据：

```
触发条件（需同时满足3个以上）：
□ 螺纹钢期货主力合约：连续站稳 3500元/吨 以上（≥4周）
□ 高炉开工率：突破 90% 并持续回升
□ 日均铁水产量：突破 250万吨/天 并维持
□ 独立焦化厂开工率：突破 80%
□ 钢厂盈利比例：回升至 50% 以上
```

### 第二层：同步确认信号（反转时同步出现）

```
触发条件（需满足2个以上）：
□ 焦炭月度提涨次数 > 提降次数
□ 焦煤流拍率：从30%+ 持续下降至 10% 以下（连续3周）
□ 焦化厂焦煤库存天数：下降至 10天 以下
□ 焦煤现货价：周环比连续3周上涨
```

### 第三层：最终确认（反转后1-2个季度）

```
□ 全行业吨焦利润：从负值回到正值且持续4周以上
□ 山西低硫主焦现货价：回到 1800元/吨 以上
□ JM期货主力：回到 1400元/吨 以上
```

## 四、止盈/止损信号

### 止盈信号（按烈度）

| 信号 | 操作 |
|------|------|
| 焦煤现货周涨幅超过10%且出现"逼仓"迹象 | 减仓1/3，顶部信号 |
| 高炉开工率从顶部回落超过3个百分点 | 减仓一半 |
| 铁水日产量见顶后连续4周下降 | 全部出清 |
| 资产注入公告落地 | 按+20%目标先减一半（你现有设定） |

### 止损信号

```
□ 个股跌破买入价 -15%（你现有设定，执行）
□ 螺纹钢价格跌破 3000元/吨（周期下行加速，退出等待）
□ 政策转向：限制焦煤出口或大规模进口补贴
```

### 5.3 周期位置评分逻辑（核心模块）

```python
# cycle_position.py 核心逻辑伪代码

def calculate_cycle_score():
    score = 0  # 0=底部，100=顶部
    
    # 价格维度（权重30%）
    rebar_vs_5yr_avg = 螺纹钢现价 / 五年均价
    jm_vs_5yr_avg = 焦煤现价 / 五年均价
    score += price_score(rebar_vs_5yr_avg, jm_vs_5yr_avg) * 0.3
    
    # 开工率维度（权重30%）
    score += normalize(高炉开工率, low=75, high=95) * 0.3
    
    # 盈利维度（权重25%）
    score += normalize(钢厂盈利比例, low=20, high=80) * 0.25
    
    # 库存维度（权重15%）
    score += normalize(螺纹钢库存, low=300, high=800, inverse=True) * 0.15
    
    return score

# 解读：
# score < 25 → 周期底部，考虑布局
# 25 < score < 50 → 复苏早期，持有
# 50 < score < 75 → 景气中期，持有+止盈准备
# score > 75 → 顶部区域，减仓
```

### 5.4 反转信号检测逻辑

```python
# reversal_signals.py 核心逻辑伪代码

def check_layer1_signals():
    """先行信号，满足3个以上触发"""
    signals = {
        "rebar_above_3500": 螺纹钢均价 > 3500 and 连续周数 >= 4,
        "bf_utilization_90": 高炉开工率 > 90,
        "hot_metal_250": 日均铁水产量 > 250,
        "coking_utilization_80": 焦化厂开工率 > 80,
        "mill_profit_50": 钢厂盈利比例 > 50,
    }
    return sum(signals.values()) >= 3, signals

def check_layer2_signals():
    """同步信号，满足2个以上触发"""
    signals = {
        "coke_raise_dominant": 月度提涨次数 > 提降次数,
        "auction_fail_low": 焦煤流拍率 < 10 and 连续周数 >= 3,
        "coking_coal_stock_low": 焦化厂库存天数 < 10,
        "spot_rising": 焦煤现货周环比上涨 连续3周,
    }
    return sum(signals.values()) >= 2, signals
```
