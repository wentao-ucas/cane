# 农产品周期研究的五层叠加分析框架

**Why**: 用户在做白糖 SR0 20 年复盘时验证了这套方法对周期性农产品非常有效。农产品有天然的"蛛网理论"周期 + 季节性供应窗口 + ENSO 气候因子，这是它和工业品/金融品最大的差异。这套方法应该成为分析任何周期性农产品的默认起手式。

**How to apply**: 当用户问任何周期性农产品（糖、棉花、玉米、橡胶、可可、咖啡、棕榈油、大豆等）的中长期分析时，按这五层顺序展开。每层都用数据/事实说话，不要直接给结论。

## 五层框架

### 第一层：价格数据基础（必做）
- 拉**日线数据 resample 到月 K**（不用直接拿月线，自己控制更可控）
- 计算每月涨跌幅、季节性（按自然月分组：均值/胜率/中位数/最大最小）
- 输出"年×月"pivot table + 热力图
- 同时拉**国际对标品种**（如白糖的 ICE 11 号、棉花的 ICE 2 号、棕榈油的 BMD 等）
- **数据源经验**：
  - 国内期货主连：akshare `futures_main_sina(symbol="SR0")` 等
  - 国际期货：**yfinance 比 akshare 的东财/新浪接口稳定得多**。SB=F (糖)、CT=F (棉花)、CC=F (可可)、KC=F (咖啡)、ZS=F (大豆) 等
  - 东财 `futures_global_hist_em` 经常被本地代理拦截，避免依赖

### 第二层：内外盘联动 + 背离分析（必做）
- 计算全样本相关系数 + 分时段相关系数
- 找出**背离最大的 N 个月**，因为背离月份通常都对应**国内独立政策事件**（收储/抛储/配额/关税/进口管控）
- 这是判断"国内品种独立性"的关键，决定能否用海外信号做国内交易

### 第三层：ENSO 滞后效应（农产品独有）
- 拉 NOAA ONI 月度数据：`https://psl.noaa.gov/data/correlation/oni.data`（文本格式，1950 至今）
- **不要算同期相关性**（接近 0），要算 **6-12 个月滞后**的相关性
- 用 **6 个月滑动平均 ONI**（消除短期噪音）+ **未来 6/12/18 月前瞻收益**
- 按 ENSO 分桶：中强 El Niño (≥1.0)、弱 (0.5-1.0)、中性、弱 La Niña (-0.5--1.0)、中强 La Niña (≤-1.0)
- 重点产品的 ENSO 敏感性：
  - **糖**：El Niño 利多（印度泰国干旱减产）
  - **棕榈油**：El Niño 利多（马来印尼干旱）
  - **可可**：El Niño 利多（西非干旱）
  - **大豆**：La Niña 利多（南美干旱减产）
  - **玉米**：La Niña 利多（美国干旱）
- 重点看 **ONI 突破 ±1.0 的关键时点** vs 后续 12 月收益（5-10 个历史突破点的胜率回测）

### 第四层：成本线 + 种植周期（决定底部信号）
- 找**完全生产成本数据**（甘蔗、棉花、棕榈果、可可豆等）
- **种植周期是关键**：
  - 一年生（玉米、大豆、棉花）：价格信号→次年改种，**周期 2-3 年**
  - 多年生宿根（甘蔗）：3-4 年宿根周期，**周期 5-7 年**
  - 永久作物（棕榈树 25 年、可可树 30 年）：**周期 10-15 年甚至更长**
- 周期越长，价格波动越剧烈、底部酝酿越久
- **底部真正信号**：不是价格便宜，而是**面积/产能开始下降**（如甘蔗的"桉进蔗退"、棕榈油的"老树砍伐率"）
- 跌穿成本线只是必要条件，**必须等到生产者真正减产兑现**才是右侧

### 第五层：5 年增减产周期定位
- 大部分农产品历史上呈现**4-6 年牛熊周期**
- 牛市顶 → 农户抢种 → 1-2 年后产量释放 → 价格崩 → 2-3 年后改种 → 减产兑现 → 新牛市
- 这是"蛛网理论"的典型表现，**周期天然长于工业品**
- 判断当前位置 = 上轮顶部多少年前 + 累计跌幅多少 + 种植面积变化 + ENSO 状态

## 输出结构（用户验证过的好格式）

1. **数据文件**：CSV 放 `data/`（月度+季节性+背离+ENSO 合并）
2. **大波动月新闻复盘**：用并行 Agent 分时段（每 5 年一组）跑 WebSearch + 写 markdown
3. **结论文档**：`output/xxx_conclusions.md`，按"季节性/内外联动/ENSO/驱动权重/周期定位/当前位置"6 段式
4. **可视化**：年×月热力图（matplotlib RdYlGn 配色）

## 几个用户验证过的写作偏好

- 用**表格对比**比文字描述更有效
- 用**Top N 极端样本**举例（而不是平均值），用户能记住
- "实操含义"段比"理论分析"段价值高
- 给出**条件概率判断**而不是"很可能/不太可能"含糊表述（如 A 情景 20% B 55% C 25%）
- 不要直接说"必然反转"，要列**触发条件清单**让用户自己跟踪

## 一句话总结

**周期农产品 = 季节性供应窗口（短期）+ ENSO 气候因子（中期 6-12 月）+ 种植周期（长期 3-7 年）+ 国内政策（独立扰动） + 全球过剩/短缺平衡（基本面定调）**
